# AI时代系统思维

> 帮助中国普通人从零进入AI世界，并逐步成长为AI系统构建者。

## 项目类型

书籍创作 + 静态网站项目

## 在线预览

[🌐 墨积成长 | InkGrowth](https://r2fdm6ncsm-prog.github.io/InkGrowth-Notes/prototype/index.html)

## 项目定位

不是工具说明书，不是编程教材，不是Prompt合集。
而是一本中国普通人的AI成长体系书籍。

## 目标用户

AI焦虑人群、想转型者、小白学习者、副业人群、内容创作者、小老板

## 核心观点

- AI不是聊天工具，而是生产力系统
- AI负责执行，人负责逻辑
- Prompt本质是任务设计
- 未来最值钱的能力，是设计智能系统的能力

## 成长路线

Level 1: AI使用者 → Level 2: AI协作者 → Level 3: AI构建者 → Level 4: AI系统设计者

## 写作规范

- 吴军式理性与系统性 + 得到系结构化表达
- 通俗、深刻、有未来感、有成长感
- 禁止学术化、教材化、纯技术语言
- 每章固定模板：问题→误区→逻辑→案例→工具→上手→总结→金句

## 作者人格

本书由 **真人作者 + AI写作搭档** 协作完成。

- 详细定义见 `doc/作者人格.md`
- 核心定位：一个先走了一步的人，回过头来给后来者画地图
- 语气坐标：高信息密度 × 温语气 × 强节奏感 × 强结构感

## 写作风格

- **结构清晰**：每个段落只完成一个功能
- **个人温度**：真实经历优先，情绪表达克制
- **信息密度**：每句话都有认知价值，不写废话
